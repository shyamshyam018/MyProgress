Sum of Numbers from 1 to N :
```
public class Main
{
    public static int sum(int n) {
        if (n <= 1) return n;  // Base case: when n is 1 or less, return n
        return n + sum(n - 1);  // Recursive case: add n to the sum of numbers from 1 to n-1
    }

    public static void main(String[] args) {
        int n = 10;
        int result = sum(n);
        System.out.println("Sum from 1 to " + n + " is: " + result);
    }
}
```

Reversing an Array Recursively :
```
public class Main {

    // Function to reverse the array recursively
    public static void reverseArray(int[] arr, int start, int end) {
        if (start >= end) {
            return;  // Base case: when the two pointers meet or cross, stop
        }
        
        // Swap the elements at start and end
        int temp = arr[start];
        arr[start] = arr[end];
        arr[end] = temp;
        
        // Recursively reverse the rest of the array
        reverseArray(arr, start + 1, end - 1);
    }

    public static void main(String[] args) {
        int[] arr = {1, 2, 3, 4, 5, 6};

        System.out.println("Original Array:");
        for (int i : arr) {
            System.out.print(i + " ");
        }

        // Call the recursive function to reverse the array
        reverseArray(arr, 0, arr.length - 1);

        System.out.println("\nReversed Array:");
        for (int i : arr) {
            System.out.print(i + " ");
        }
    }
}
```

Check if a String is Palindrome or not  :
```
public class Main {

    // Recursive function to check if the string is palindrome
    public static boolean isPalindrome(String str, int start, int end) {
        // Base case: if start meets or crosses end, the string is a palindrome
        if (start >= end) {
            return true;
        }

        // If characters at start and end are not the same, it's not a palindrome
        if (str.charAt(start) != str.charAt(end)) {
            return false;
        }

        // Recursive case: move towards the middle
        return isPalindrome(str, start + 1, end - 1);
    }

    public static void main(String[] args) {
        String str = "madam";  // You can test with different strings here

        // Call the recursive function
        if (isPalindrome(str, 0, str.length() - 1)) {
            System.out.println(str + " is a palindrome");
        } else {
            System.out.println(str + " is not a palindrome");
        }
    }
}
```

Find First Occurring UpperCase :
```
public class Main {

    // Recursive function to find the first uppercase letter
    public static char findUpperCase(String str, int index) {
        // Base case: If we've gone through all characters
        if (index == str.length()) {
            return '\0'; // No uppercase letter found, return null character
        }

        // If current character is uppercase, return it
        if (Character.isUpperCase(str.charAt(index))) {
            return str.charAt(index);
        }

        // Recursively check the next character
        return findUpperCase(str, index + 1);
    }

    public static void main(String[] args) {
        String str = "shYaM";
        char result = findUpperCase(str, 0);  // Start with index 0

        if (result != '\0') {
            System.out.println("First uppercase letter is: " + result);
        } else {
            System.out.println("No uppercase letter found.");
        }
    }
}
```

Check if the number is Palindrome or not :
```
public class Main {
    // Recursive function to reverse the number
    public static int reverseNumber(int n, int rev) {
        // Base case: when n becomes 0, return the reversed number
        if (n == 0) {
            return rev;
        }
        
        // Extract last digit, and make recursive call with reversed number
        return reverseNumber(n / 10, rev * 10 + n % 10);
    }

    public static void main(String[] args) {
        int n = 121;  // Example number
        int reversed = reverseNumber(n, 0);  // Initial reversed number is 0
        
        // Compare original number with the reversed number
        if (n == reversed) {
            System.out.println(n + " is a palindrome.");
        } else {
            System.out.println(n + " is not a palindrome.");
        }
    }
}
```
